HTML Boilerplate Code

This is the basic starting structure of every HTML page.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

File Path in HTML

File paths tell the browser where to find files (images, videos, links).

1. Same Folder

Image is in the same folder as HTML file:

<img src="image.jpg" alt="Sample">

2. Subfolder

Image inside an images folder:

<img src="images/photo.png" alt="Photo">

3. Parent Folder

Image is one folder back (.. means parent directory):

<img src="../photo.png" alt="Parent Photo">

4. Absolute Path

Full web address:

<img src="https://example.com/logo.png" alt="Logo">
