Concept of hyperlink

A hyperlink is a clickable text or image that links to another resource. Use the <a href="..."> tag.

What is HTML (HyperText Markup Language)?

HTML is a markup language used to structure content on the web. Tags like <h1>, <p>, <a>, <img> are basic building blocks.
