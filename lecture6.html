1. Media Tags in HTML

Media tags are used to add images, audio, video, and external content.

Image Example:

<img src="photo.jpg" alt="Sample Photo" width="300">

Audio Example:

<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>

Video Example:

<video width="300" controls>
  <source src="video.mp4" type="video/mp4">
</video>

Iframe Example:

<iframe src="https://www.wikipedia.org" width="400" height="200"></iframe>

2. Multi Page Website

A multi-page website is made of different HTML files like index.html, about.html, gallery.html, contact.html. These pages are linked using the <a> tag.

Navigation Example:

<a href="index.html">Home</a> |
<a href="about.html">About</a> |
<a href="gallery.html">Gallery</a> |
<a href="contact.html">Contact</a>

3. Deploy Your Project

Deployment means putting your project online so anyone can see it.

Method 1: GitHub Pages

  1. Create a GitHub account and repository.
  2. Upload all your HTML files.
  3. Go to Settings → Pages → select branch.
  4. Your site will be live at https://username.github.io/repository-name/

Method 2: Netlify

  1. Go to Netlify.
  2. Sign up and log in.
  3. Drag & Drop your project folder into Netlify dashboard.
  4. It gives you a live link like https://myproject.netlify.app

Summary

  • Media Tags → Add images, audio, video, iframe.
  • Multi Page Website → Multiple HTML files linked with <a>.
  • Deploy Project → Upload to GitHub Pages or Netlify to make it live.
