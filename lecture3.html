Nested Lists & Tables — Complete Guide

Examples and explanations of HTML lists and tables. (No CSS used.)

1. Nested Lists

1.1 Unordered List (nested)

An unordered list can contain another <ul> as a child of an <li>:

      • Orange
      • Lemon

Ordered lists can be nested too; the inner list can be ordered or unordered.

1.5 Common accessibility tip for lists

Keep list items short and use proper nesting: the child list must be inside an <li> element.

2. Tables — Structure & Examples

Tables present tabular data. Use <caption>, <thead>, <tbody> and <tfoot> for structure and accessibility. Use scope for header cells to help screen readers.

2.1 Simple table with thead / tbody / tfoot

Monthly Sales (Example)

January 120 80 200

Quarter Total 720

2.2 Table using colspan and rowspan

colspan merges cells horizontally; rowspan merges vertically.

2.4 Accessibility & semantics tips for tables

  • Use <caption> to describe the table briefly.
  • Use <thead>, <tbody>, <tfoot> to group rows logically.
  • Use <th scope="col"> for column headers and <th scope="row"> for row headers to help screen readers.
  • Prefer not to use tables for layout; use them for tabular data only.

4. Quick checklist before using lists & tables

  • Use lists for sequences or grouped items; use tables for data with rows & columns.
  • Keep nested lists readable — don't nest too deeply unless necessary.
  • Give tables captions and use semantic sections (<thead>, <tbody>, <tfoot>).
  • Remember accessibility attributes: scope on <th> and clear captions.
